Hey ...

Die Bremen.js ist eine User Group für und von Enthusiasten, die die Webentwicklung mit JavaScript außerordentlich “awesome” finden. Sie zollt dem Tribut, womit sich jeder “JavaScript Hacker” täglich konfrontiert sieht: einer außergewöhnlichen Community. Da dieser Austausch in der Regel nun einmal über das Web stattfindet, bietet die Bremen.js eine regionale Plattform um somit mal die Gesichter hinter dem virtuellen Avatar kennenzulernen. Dabei steht das Networking und der Wissensaustausch über Themen aus dem Dunstkreis der Webentwicklung mit JavaScript im Vordergrund. Wenn ihr also Node.js kennt aber nicht genau wisst, ob diese Technologie für das nächste Projekt sinnvoll ist, ihr bisher mit jQuery gearbeitet habt und nun “richtiges” JavaScript lernen möchtet oder beispielsweise die Idee hinter NoSQL-Storage-Systemen interessant findet, so werdet ihr an jedem zweiten Montag des Monats bei der Bremen.js im Rahmen von Vorträgen und Diskussionen mit Sicherheit den ein oder anderen Aspekt bzw. Kontakt mitnehmen können.

Eingeladen ist jede/jeder Interessierte. Egal, ob Webentwickler oder Designer, Backend- oder Frontend-Entwickler, Hacker oder Newcomer; jeder Input ist wertvoll. Ihr seid alle willkommen.

Wenn ihr Anregungen habt, mitorganisieren wollt oder auch Vorträge halten möchtet, so schreibt uns einfach via Twitter, Google+ oder postet auf unserer Mailingliste.

Ihr seid Bremen.js - Wir freuen uns auf euch.
